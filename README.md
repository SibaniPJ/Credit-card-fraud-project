# Credit-card-fraud-project
The Cyber Fraud Detection Dashboard provides an interactive visual analysis of fraudulent transactions across multiple dimensions. It helps in understanding the volume, distribution, and patterns of fraud, enabling better monitoring and decision-making in cybersecurity and finance-related risk management.

##Project Purpose
Cyber fraud activities have increased significantly with the growth of digital transactions.
The goal of this dashboard is to provide:
        Fraud Identification
        Behavioral Patterns between fraudulent and legitimate transactions
        High-risk customer/transaction insights
        Trends & fraud movement across time and geography

##Dataset
<a href="https://github.com/SibaniPJ/Credit-card-fraud-project/blob/main/credit_card_fraud_dataset.csv">Dataset</a>

##Data Source
Dataset includes the following features:
             Transaction ID, Date & Time
             Amount
             Location / City
             Customer ID / Account details (anonymized)
             Transaction Type
             Fraud Status (Fraud / Non-Fraud)

##Key Visualizations & Features
KPI Summary Cards: Total Transactions, Fraud Count, Fraud Rate, Total Fraud Amount
             Fraud vs Non-Fraud Comparison Chart
             Fraud Cases Over Time Line Graph
             Fraud Distribution by Location (Map View)
             Transaction Type Breakdown
             Filter Controls for:
             Transaction Type
             Date Range
             Region / City
             Fraud/Non-Fraud

##Insights Derived
Fraud transactions usually occur in higher amounts compared to normal transactions.
Certain locations have more fraud density indicating hotspot zones.
Fraud shows noticeable spikes at specific times (time-based patterns).
Transaction Type difference clearly separates normal vs fraud behavior.
